Here is an example of a setup with multiple players. When one of them starts, the others are automatically paused:

 
 
 

Embed code

As always, ensure you have uploaded a copy of JW Player and included the script in the <head> of your page, together with your player key. See the Quick Start Guide for more info. Next, place the following embed code at the location you want the players to appear:

<div id="player1"></div>
<div id="player2"></div>
<div id="player3"></div>

<script>
var playerInstance1 = jwplayer("player1");
var playerInstance2 = jwplayer("player2");
var playerInstance3 = jwplayer("player3");
playerInstance1.setup({ file: "/uploads/myAudio.aac", height: 40 });
playerInstance2.setup({ file: "/uploads/myAudio.aac", height: 40 });
playerInstance3.setup({ file: "/uploads/myAudio.aac", height: 40 });
</script>

Second, add the JavaScript to implement the logic for pausing other players when one starts:


  playerInstance1.on('play',function(){ pausePlayers(1); });
  playerInstance2.on('play',function(){ pausePlayers(2); });
  playerInstance3.on('play',function(){ pausePlayers(3); });

  function pausePlayers(playing) { 
    for (var i=1; i<4; i++) {
       if(i != playing) { jwplayer("player"+i).pause(true); }
    }
  };

This script uses the playerInstance.on('play'); call to find out which player just started. Next, the playerInstance.pause(true) call is used to force all other players to pause. See Using the JavaScript API for more info.

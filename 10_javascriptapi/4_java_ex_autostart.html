This feature is built using the JW Player JavaScript API, with just a few lines of JavaScript code.

Embed code

Assuming that you have a functional JW Player setup as mentioned here, you'll need to add the following code to your page:

<script>
  if(document.cookie.indexOf("jwplayerAutoStart") == -1) {
    document.cookie = "jwplayerAutoStart=1";
    playerInstance.play();
  }
</script>

This script uses a cookie to store that a viewer was autostarted before. If that's not the case, the playerInstance.play() function is called. However, please be aware that autostart does not work on mobile devices.

This article applies to users of the JW Platform dashboard who have or are planning to migrate to the new dashboard.

In the new JW Player dashboard, you will notice a brand new design, and several new features including:  

  • Deeper Analytics: Including device, domain & country breakdowns.
  • External URLs: Manage, embed & analyze self-hosted (live) streams.
  • Multiple Users: Manage access for more users in admin or editor roles.
  • Player Downloads: Get the self-hosting ZIPs, including our mobile SDKs.

Moving to our new dashboard will not affect your content delivery, in-progress transcodes, API integrations or billing. Only the user interface will be updated (instantly and seamlessly). Watch the video demo of the dashboard for a helpful walk-through.
 

Dashboard Homepage

Links to access important tools:

  • Player + SDK downloads and license keys
  • Blog posts about latest features and updates
  • Support Articles and link to Contact Support


 
Quick Publish Module
Drag and drop single files to quickly upload, or add by URL.  (If you need to add files in bulk, do that via the content list page.)


 


Publish Mode

On pages from which you can embed players or content, you will notice the Publish Mode widget at the top of the page.  Each “Embed” link or button will open Publish Mode to create your single line embed code.  Additionally, if you are embedding a single video, you can choose to include schema.org SEO tags which automatically pulls from your video metadata.  You can also use Publish Mode to preview content using the preview page link.   


 

Content: Video Library and Playlists

Under the content section you will find your video library and your playlists.  You can bulk add videos (or audio) by uploading files or adding external URLs.  

Content Detail

Add tags, metadata, tracks and view sources for your video.  Within the tracks tab, add and edit caption files using our NEW captions editor tool.  

Analytics

Monitor your content performance, and look at activity by device, geography, engagement, and more. Player activity is associated with a unique media ID, to ensure that everything is attributed accurately. Read more about Media IDs, using the new analytics, and most importantly what's new in Analytics as compared to your JW Platform dashboard.


   

Account: Manage Multiple Users

In the Account menu you will find access to:

  • Users - add and edit account users and permissions
  • API Keys
  • Billing information
  • Usage analytics
 

 

Features: Coming soon!

The following Platform dashboard features are not yet available in the new JW Player dashboard. Refer to the linked documentation so you can configure these features via the Platform API in the mean time while we are still working on building these in the new dashboard:

  • Geo-blocking
  • Content or player signing
  • Scheduling content to be published later (aka content sunrise)

There are instances when the JW Player's default text may need to be altered. A good case for this would be if you need to localize JW Player for a non-English speaking market. A reference of all customizable text is below.

General Playback

  1. title or displaytitle: Either can change the title attached to the play button, however displaytitle will override any titles set in a playlist. This will also affect the text appearing in the control bar during a live stream.
  2. abouttext: Customizes the right click text.

Advertising

When using our VAST client, there are three items that are able to be customized. Each of these options would be configured in your advertising block. Please note that all of these customization options are not available when using our Google IMA client. Currently only admessage is supported.



 
  1. admessage: The message that will display as an ad plays. See here for more information.
  2. skipmessage: The text that displays before an ad can be skipped
  3. skiptext: The text that appears after an ad can be skipped
  4. cuetext: The text that shows up when an advertising cuepoint is moused over.

Sharing and Related Tooltips

Text used within our Sharing and Related features is not directly editable, however the tooltips that appear when a mouse hovers above the buttons can be customized with the heading option. Each is configured within the appropriate related or sharing block. Default text for each can be seen below.

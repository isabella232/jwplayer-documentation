This document only applies to JW Player Enterprise users. Don't have Enterprise or our Android SDK?
Upgrade Now.

In order to take full advantage of our SDK improvements, you should update your current Android development environment using the following instructions:

  • Sign in to your JW Player Dashboard and download the latest SDK version from your Account tab.

  • In Android Studio, remove the module dependency for jwplayer-android-x.y.z for your app, then remove the jwplayer module itself.

  • Import the new .AAR file contained in the newest version Zip package downloaded from your Account tab in the JW Player Dashboard. Finally, add the new jwplayer-android-x.y.x module as a module dependency for your app.
     

If you are using Maven:

  • If the dependency version is a + (e.g. compile 'com.longtailvideo.jwplayer:jwplayer-android-sdk:+') in build.gradle the SDK will update automatically when each build is done.

  • If there is a locked-in version (e.g. compile 'com.longtailvideo.jwplayer:jwplayer-android-sdk:1.1.99') you must change it to the latest version number or a plus sign.

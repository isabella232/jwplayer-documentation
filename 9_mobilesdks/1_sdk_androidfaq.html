The JW Player SDK for Android has media classes and methods that enable developers to build JW Player directly into any Android application, expanding your content reach to include a wider mobile audience.


Q: How do I get the Android SDK? How do I use the Android SDK?

A: Please use our contact form so we can work with you to understand your requirements.


Q: Is the JW Player Android SDK included with my Enterprise license?

A: No, the Android SDK is priced separately and is considered a different product. Use our contact us form to reach out to a friendly JW Player representative for more pricing information


Q: I've purchased the SDK software, but where do I download it?

A: If you have an SDK license, a download link will appear in your JW Player account dashboard. If you're not seeing a download link, please contact your JW Player account representative.


Q: When I create a JWPlayerView and add a source, I get an error "License Key Not Found".

A: Either you do not have a JW product key, or the key you have is not an Enterprise key. Contact support@jwplayer.com to validate your key.
Also, make sure the element is inside the closing tag in your project’s AndroidManifest.xml file.


Q: When I launch my emulator, I see all kinds of errors and strange behavior.

A: IMPORTANT: The Android Media Player is known to be unstable when running in Android emulators, often throwing "Unknown Android MediaPlayer error (extra = -2147483648)." We strongly recommend using physical devices to develop and test apps you build with the JW Player SDK.


Q: The demo app is buggy/tricky to use.

A: The sample app is intended only to allow for a quick and easy way to see a number of streaming and progressive download video playback examples, with and without VAST based video advertising, and to illustrate how the API in the library can be used.


Q: When building my app, I get the error:
Could not GET https://mvn.jwplayer.com/content/repositories/
releases/com/longtailvideo/jwplayer/
jwplayer-android-sdk/'. Received status code 401 from server: Unauthorized.

A: You have entered an incorrect username or password for the JW Player maven repository in your AndroidManifest.xml file. Contact support@jwplayer.com for correct login credentials.


Q: Can I get the source code for the demo app?

A: The demo application source code for our 2.x Android SDK now lives on GitHub at:
https://github.com/jwplayer/jwplayer-sdk-android-demo


Q: Can I try my own video URL in the demo app?

A: Yes, all you need to do is configure your PlaylistItem object.


Q: How do I remove the demo app from my device?

A: To uninstall the demo app, long-tap the JW Player Mobile icon in the app drawer, and drag it to the top of the screen to the Uninstall bar.

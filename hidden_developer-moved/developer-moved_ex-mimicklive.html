In this example, we'll be taking a look on how to mimic live TV!

​

If this page is refreshed, you'll notice that the video will start up right where it left off. The trick to getting this to work is with our seconds variable here:

var seconds = new Date().getMinutes()*60 + new Date().getSeconds();

​When the page is loaded, we will seek to a point in the video files designated by the "seconds" variable. (This utilizes the seek() API call. For a full list of API calls, please see our page here.) As long as users' devices are set to the proper time, they should all be synchronizing to the proper point in the video file. Also in our code, we will be cycling through playlist items to determine the proper file that should be played. Again, this is determined based on the seconds var.

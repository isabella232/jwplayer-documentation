​​VAST Advertising

The JW Player receiver application is able to playback Linear VAST prerolls, midrolls, and postrolls. The following VAST 3.0 playback features are supported while casting:

  • Ad pods
  • Skippable ads
  • Companion ads
This document applies to users of JW Player Ads. Don't have Advertising? Upgrade Now.

VPAID advertising creatives and nonlinear ads are not supported during a casting session because they cannot be played on a Chromecast. When an ad is being played, the countdown message and video click-through link is sent back to the sender application in the Chrome browser.

Clicking anywhere except the controls of the Player will pause the video on the receiver and take the viewer to the VideoClick link provided in the VAST XML. This action does not fire a VAST clickthrough tracker but it does fire the onAdClick API event.

Advertising API

There is no support for the Javascript Advertising API at this time.

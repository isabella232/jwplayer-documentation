In order to take full advantage of our iOS SDK improvements, you should update your JW Player framework within Xcode using the following instructions:

  • Overwrite the existing “JWPlayer-iOS-SDK.framework” file linked in your project with the new framework version.

  • Build and run your project.

If you still have questions, check out our SDK FAQ or the iOS SDK section of our developer portal.
